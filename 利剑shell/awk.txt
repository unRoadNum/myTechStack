## awk

用途
***
使用awk进行高级文本处理，逐行处理

基本用法
***
awk 'BEGIN { print "start"} pattern { print} END { print "end"}' file

注解
***
1. BEGIN和END语句块都可以省略
2. pattern省略后，默认为对每一行处理， 且默认处理语句为{ print }
	* 也就是说上述就算不写print也一样
3. 

使用举例
***
echo -e "line1\nline2" | awk 'BEGIN { print "Start"} { print} END { print "End"}'
