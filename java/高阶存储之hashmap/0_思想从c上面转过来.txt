#思想从c上面转过来#

因为写惯了c代码，你要一时半会从c上面的思想转过来也是不容易的。
你什么都想亲力亲为，像个从农村来的独立自强的小伙儿。
这一点在数据存储上尤为明显。

#这里的数据存储更像是在分配这一层上的逻辑抽象#，因为你使用的永远不可能是字节流。
你得对它编码让它变成整型、浮点型、字符型，甚或者是数组、指针。
无论是临时存储还是全局变量静态存储，或者在堆上分配，
我们操作的永远是它上面的整型、浮点型、字符型等等。
这一点在 HELLO_STRING_T *p = (HELLO_STRING_T*)malloc(sizeof(HELLO_STRING_T))也可以看出来。

而java比c做的更多，它除了提供给你基本类型之外，还有更加好用的Collection类。
你可以直接使用它，像在c中使用数组一样，而不用维护和构建它，甚或者释放它。

那么我就从hashMap开撸。
