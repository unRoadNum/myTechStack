#java基本类型使用#

统计起来，java有8种基本类型： 6种数字类型、1种字符类型和1种布尔型。

数字类型：
1. byte： 8位 有符号的整数  -128到127
	* 主要用在数组中节约空间，主要代替整数，它只有int的1/4
2. short: 16位  有符号整数 -32768到32767
3. int：32位 有符号整数  -2的31次方 到 2的31次方减1
4. long：64位 有符号整数 -2的63次方 到 2的63次方减1
***
5. float: 32位 单精度浮点数
6. double： 64位 双精度浮点数

字符类型：
char：16位  unicode字符

布尔型：
boolean：单个的boolean 类型变量在编译的时候是使用的int类型。
而对于boolean 类型的数组时，在编译的时候是作为byte array来编译的所以boolean 数组里面的每一个元件占一个字节，这是确定的

#没有必要去记住这些范围的#
只要使用包装类就可以查询，比如: Byte.SIZE  Byte.MIN_VALUE	Byte.MAX_VALUE
